# Healing Map Dashboard

React-based frontend for the Post-Flood Recovery Analysis platform.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

The dashboard will open at `http://localhost:3000`

## Features

- Interactive healing map with Leaflet
- Recovery metrics visualization
- Time-series charts
- Event processing interface

## Configuration

Set `REACT_APP_API_URL` in `.env` to point to the backend API (default: `http://localhost:5000/api`)
